# تطبيق معدلي - حساب المعدلات

تطبيق ويب تقدمي (PWA) لحساب المعدلات لطلبة:
- العلوم الإنسانية
- العلوم الاجتماعية
- جامعة سطيف 2

## المميزات
- ✅ حساب المعدل التلقائي
- ✅ العمل دون اتصال بالإنترنت
- ✅ إمكانية التثبيت على الهاتف
- ✅ واجهة باللغة العربية

## كيفية الاستخدام
1. اختر المستوى الدراسي
2. اختر السداسي
3. أدخل العلامات
4. اضغط على "احسب المعدل"
